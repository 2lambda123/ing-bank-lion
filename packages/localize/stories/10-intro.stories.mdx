# Localize

Is meant to translate text into multiple languages.
In it's simplest form it is a function that returns the translated text for a namespace + key.

## Features

- Uses es modules
- Formatting of numbers, amounts
- Formatting of dates

## Content

| Feature                                                               | Description                                            |

| [Translate Text](?path=/docs/buttons-button--default-story)           | Load and translate text in multiple languages          |
| [Web Components](?path=/docs/buttons-button--default-story)           | Use translations within a web component                |
| [Format Numbers](?path=/docs/buttons-button--default-story)           | Format numbers in multiple languages                   |
| [Format Dates](?path=/docs/buttons-button--default-story)             | Format dates in multiple languages                   |

## How to use

### Installation

```sh
npm i --save @lion/localize
```

```js
import { localize } from '@lion/localize';
```
