## How to use

### Installation

```sh
npm i --save @lion/icon
```

```js
import '@lion/icon/lion-icon.js';
```

### Example

```js
import './somewhere/my-iconset.js';
```

```html
<lion-icon icon-id="somewhere:myIconset:foo"></lion-icon>
```
